Zen Browser DNF Repository

Fedora/RHEL/CentOS packages for Zen Browser

Note: This repository is automatically updated when new Zen Browser releases are available.

1. Add the Repository

Create the repository configuration file:

sudo tee /etc/yum.repos.d/zen-browser.repo > /dev/null << 'REPO'
[zen-browser]
name=Zen Browser
baseurl=https://qomarhsn.github.io/zenbrowser-dnfrepo/
enabled=1
gpgcheck=0
repo_gpgcheck=0
skip_if_unavailable=True
metadata_expire=6h
REPO

2. Install Zen Browser

sudo dnf clean all
sudo dnf install -y zen-browser

3. Update Zen Browser

To update to the latest version:

sudo dnf upgrade -y zen-browser

Package Information

  • Installation Path: /opt/zen-browser
  • Binary Symlink: /usr/bin/zen
  • Launcher: /usr/bin/zen-browser
  • Supported Architectures: x86_64
